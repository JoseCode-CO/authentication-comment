Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

A function that allows you to authenticate a user.

Example request:
curl --request POST \
    "http://localhost/api/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"jose123@gmail.com\",
    \"password\": \"root\"
}"

const url = new URL(
    "http://localhost/api/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "jose123@gmail.com",
    "password": "root"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Store a newly created resource in storage.

Example request:
curl --request POST \
    "http://localhost/api/v1/messages" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "sender_id=molestiae" \
    --form "receiver_id=voluptatibus" \
    --form "post_id=voluptatem" \
    --form "message=minus" \
    --form "attachments=@C:\Users\Acer\AppData\Local\Temp\php4F9E.tmp" 
const url = new URL(
    "http://localhost/api/v1/messages"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('sender_id', 'molestiae');
body.append('receiver_id', 'voluptatibus');
body.append('post_id', 'voluptatem');
body.append('message', 'minus');
body.append('attachments', document.querySelector('input[name="attachments"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());
